About Invoxi

Simple Invoice Generator — fast, free, privacy-friendly.

What is Invoxi?

Invoxi is a lightweight web app to create professional invoices in seconds and export them as PDF. No signup required, and your data stays in your browser.

Main features

  • PDF export that looks great on print
  • Optional logo, taxes (VAT), discount & delivery
  • CSV import / export
  • Signature field
  • Multi-language UI (e.g. EN/AR/FR)
Start New Invoice

Why Invoxi?

  • Speed: create an invoice in under a minute
  • Privacy: no server storage, everything stays on your device
  • Free: use it without any subscription
